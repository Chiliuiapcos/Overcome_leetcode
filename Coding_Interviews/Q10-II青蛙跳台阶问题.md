# Q10 - II：青蛙跳台阶问题

一只青蛙一次可以跳上1级台阶，也可以跳上2级台阶。求该青蛙跳上一个 n 级的台阶总共有多少种跳法。

*答案需要取模 1e9+7（1000000007），如计算初始结果为：1000000008，请返回 1。*

```cpp
示例 1：
    输入：n = 2
    输出：2
	
示例 2：
    输入：n = 7
    输出：21
```



### 思路：斐波那契数列

*n级台阶的不同跳法的总数 f(n)=f(n-1)+f(n-2)*

```cpp
class Solution {
public:
    int numWays(int n) {
        int dp[101];
        dp[0]=1;
        dp[1]=1;
        if(n<2)
            return dp[n];
        for(int i=2;i<=n;i++){
            dp[i]=(dp[i-1]+dp[i-2])%1000000007;
        }
        return dp[n];
    }
};
```
